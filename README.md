Campus360 🎓
Campus360 is a student-focused web platform designed to manage and track campus activities in one place. It provides a centralized dashboard for students, faculty, and administrators to access academic and campus-related services efficiently.

Features
Student Dashboard
Faculty Dashboard
Admin Dashboard
Attendance Tracking
Timetable Management
Course & Fee Management
Notifications
Canteen & Gymkhana Services
Lost & Found System
Profile & Settings
🛠️ Technologies Used
HTML
CSS
JavaScript
Git & GitHub
📁 Project Structure
index.html – Landing page
login.html – Login system
student-dashboard.html – Student panel
faculty-dashboard.html – Faculty panel
admin-dashboard.html – Admin panel
timetable.html – Schedule management
fees.html – Fee management
assets/ – Images and resources
🌐 Purpose
Campus360 aims to simplify campus management by providing a single platform for academic tracking, services, and student engagement.
