# Campus360 🎓

Campus360 is a student-focused web platform designed to manage and track campus activities in one place.
It provides a centralized dashboard for students, faculty, and administrators to access academic and campus-related services efficiently.

---

## 🚀 Features

* Student Dashboard
* Faculty Dashboard
* Admin Dashboard
* Attendance Tracking
* Timetable Management
* Course & Fee Management
* Notifications
* Canteen & Gymkhana Services
* Lost & Found System
* Profile & Settings

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript
* Git & GitHub

## 🌐 Purpose

Campus360 aims to simplify campus management by providing a single platform for academic tracking, campus services, and student engagement in a modern and organized way.
